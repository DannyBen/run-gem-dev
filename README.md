Run Gem Dev
===========

[Runfile](https://github.com/DannyBen/runfile) tasks for gem developers.

## Install

	$ gem install run-gem-dev

## Usage

Put this in your Runfile

```ruby
RunGemDev::Tasks.embed 'your-gem-name'
```
